# 想法记录模板

以后每次冒出一个点子，都用同样的格式记录，这样 Codex 更容易帮你拆解。

## 模板

- 名称：
- 想解决什么问题：
- 给谁用：
- 输入是什么：
- 输出是什么：
- 最小可行版本：
- 以后可以再加什么：
- 备注：

## 示例 1：待办工具

- 名称：今天要做什么
- 想解决什么问题：我总是忘记当天最重要的 3 件事
- 给谁用：自己
- 输入是什么：待办标题、截止时间、优先级
- 输出是什么：一个可以勾选完成的任务列表
- 最小可行版本：新增、删除、勾选任务
- 以后可以再加什么：筛选、分类、数据保存
- 备注：适合练习表单和状态管理

## 示例 2：灵感收集页

- 名称：灵感盒子
- 想解决什么问题：想到什么很快就忘
- 给谁用：自己
- 输入是什么：一句话灵感
- 输出是什么：可搜索的灵感列表
- 最小可行版本：保存、查看、删除
- 以后可以再加什么：标签、排序、导出
- 备注：适合练习本地存储

## 示例 3：AI 文案助手

- 名称：一句话助手
- 想解决什么问题：把零散想法变成更清楚的文字
- 给谁用：自己
- 输入是什么：主题、语气、长度
- 输出是什么：摘要、文案、建议
- 最小可行版本：一个输入框 + 一个输出框
- 以后可以再加什么：历史记录、模板、更多场景
- 备注：适合练习 API 调用
